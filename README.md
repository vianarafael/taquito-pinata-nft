# taquito-pinata-nft

## you'll need to generate a [pinata api key](https://www.pinata.cloud/).

Add in a file called PinataKeys.js and add the following:
```
module.exports = {
  apiKey: "the_key",
  apiSecret: "the_pw",
};
```

## run the:
* back-end/
```
node index.js
```
* front-end/
```
yarn start
```

## Wallet
Click on the `Connect your wallet` button and choose your prefered wallet
